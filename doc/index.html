opencv-0.0.0: Haskell binding to OpenCV-3.1

This is a Haskell library providing a binding to OpenCV-3.1. It binds directly with the C++ API using the inline-c Haskell library.

The library is far from complete but the framework is there to easily bind missing functionality.
